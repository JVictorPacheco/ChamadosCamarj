# 📦 Obsidian — ChamadosCamarj

## Como abrir no seu celular (iPhone)

Como você está no Windows agora, o jeito mais fácil é:

**1.** Baixa o arquivo `obsidian-chamados.tar.gz` do GitHub
**2.** Extrai no Windows (7-Zip ou WinRAR) → vai gerar a pasta `obsidian/`
**3.** Manda a pasta **completa** pro seu celular por:
   - Google Drive / OneDrive
   - Email
   - Cabo USB

**4.** No iPhone, abre o Obsidian → **Create new vault** → escolhe "Open folder as vault" → seleciona a pasta que você copiou

> ⚠️ **Alternativa**: Como você tem o repositório no GitHub, no celular você pode:
> - Abrir o Obsidian no iPhone
> - Usar um plugin de sincronização (Git)  
> - Ou simplesmente esperar — quando chegar no Mac em casa é mais fácil!
